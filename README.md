# 🍽️ Online Ordering Canteen System — Unilever

A Business Analysis capstone project designed by **Team Agile** (April 2023 Cohort), proposing a web-based canteen ordering system to improve the employee dining experience at Unilever's UK offices.

---

## 📄[View the full project PDF](Agile Team/docs/TEAM_AGILE_CAPSTONE_PROJECT.pdf)


## 📌 Project Overview

Unilever's existing canteen system was plagued by long queues, limited food choices, overcrowding, and significant food wastage. This project proposes a **custom-built Online Canteen Ordering System (UCOS)** to streamline the entire lunch ordering and delivery process for approximately 1,500 employees.

---

## 🚨 Problem Statement

- Monthly food wastage (~25% of inventory)
- Employees spending up to 35 minutes queuing for food and a table
- Overcrowded canteen during lunch hours
- Limited and often unavailable meal variety
- Long turnaround time reducing effective work hours

---

## 🎯 Business Objectives

| Objective | Target | Timeline |
|-----------|--------|----------|
| Reduce food wastage | Minimum 30% reduction | 6 months post-launch |
| Reduce canteen operating costs | 15% reduction | 12 months post-launch |
| Increase effective work time | +30 minutes/employee/day | 3 months post-launch |
| Reduce required canteen manpower | Via automation & desk delivery | Ongoing |

---

## ✅ Proposed Solution

A **custom-built web application** that allows employees to:

- Register using their staff ID (linked to payroll for salary deductions)
- Browse a daily updated menu with prices
- Place and edit orders before an 11:00 AM cutoff
- Choose between canteen dining or desk delivery
- Receive order confirmation notifications
- Submit feedback on food quality and delivery

### Key Features
- User-friendly interface
- Unified ordering system
- Menu management (Canteen/Menu Manager)
- Delivery request & tracking
- Feedback system
- User account management
- Automated payroll deduction integration
- Security and privacy controls

| # | Role | Story |
|---|------|-------|
| 1 | Employee | Create an account using staff ID |
| 2 | Employee | Login and edit profile |
| 3 | Employee | View daily menu with prices |
| 4 | Employee | Place an order |
| 5 | Employee | Request desk delivery |
| 6 | Employee | Provide feedback on food and delivery |
| 7 | Canteen Manager | View and manage daily orders |
| 8 | Menu Manager | Create and update daily menu |
| 9 | Canteen Manager | Close orders before 11 AM |
| 10 | Employee | Receive order confirmation |
| 11 | Employee | Enrol for salary payment deduction |
| 12 | Payroll System | Calculate total dishes ordered per employee |
| 13 | Payroll System | Make accurate salary deductions |
